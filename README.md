# Machine Learning & Data Science Portfolio

Welcome to my machine learning repository! This collection of Jupyter Notebooks documents my practical exploration of AI, data science, and machine learning concepts. As a computer engineering student, I created this space to implement, experiment with, and understand foundational algorithms in data preprocessing, regression, classification, and clustering.

## 📁 Repository Structure

The notebooks in this repository are categorized by their primary machine learning application:

### 1. Data Preprocessing
Before training any model, data needs to be cleaned, encoded, and scaled correctly.
* `data_preprocessing_tools.ipynb`: Covers essential techniques for handling missing data, encoding categorical variables, splitting datasets into training and testing sets, and feature scaling.

### 2. Regression Models
Algorithms used for predicting continuous numerical values.
* `multiple_linear.ipynb` & `multiple_linear_regression.ipynb`: Implementations of Multiple Linear Regression to predict outcomes based on several independent variables.
* `polynomial_regression.ipynb`: Models non-linear relationships between independent and dependent variables.
* `decision_tree_regression.ipynb`: Predicting continuous values using decision tree logic and node splitting.
* `random_forest_regression.ipynb`: An ensemble learning method using multiple decision trees to improve prediction accuracy and control over-fitting.

### 3. Classification Models
Algorithms used for predicting discrete categories or classes.
* `logistic_regression.ipynb`: A fundamental algorithm for binary classification.
* `k_nearest_neighbors.ipynb` (K-NN): Classifying data points based on the proximity to their *K* nearest neighbors.
* `support_vector_machine.ipynb`: Standard Support Vector Machine for finding the optimal linear hyperplane to separate classes.
* `kernel_svm.ipynb`: Support Vector Machines utilizing kernel tricks (like RBF) for non-linear classification boundaries.
* `naive_bayes.ipynb`: A probabilistic classifier based on Bayes' theorem, assuming independence between predictors.
* `decision_tree_classification.ipynb`: Using tree-based splits to categorize data into distinct classes.
* `random_forest_classification.ipynb`: An ensemble method that operates by constructing a multitude of decision trees at training time for highly accurate classification.

### 4. Clustering (Unsupervised Learning)
Algorithms used for finding natural groupings in unlabeled data.
* `k_means_clustering.ipynb`: Partitioning data into *K* distinct clusters based on feature similarity and centroids.

## 🛠️ Technologies & Libraries Used

* **Python 3**
* **Jupyter Notebook** / Google Colab
* **NumPy**: For numerical computations and array manipulations.
* **Pandas**: For importing and managing datasets.
* **Matplotlib**: For plotting and visualizing the data and model boundaries.
* **Scikit-Learn**: The core library used for building, training, and evaluating the machine learning models.
